# ProyectoFullStack
Instrucciones de instalación. 
1. Instala Node.js 20+ (incluye npm).
2. Clona el repo y ábrelo en VS Code.
3. Instala dependencias: npm install (ctrl + ñ para la consola en vs code).
4. Levanta el backend antes de probar.
5. Ejecuta en desarrollo: npm run dev en la consola y abre la URL que muestra Vite (http://localhost:5173).
